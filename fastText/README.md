## Requirements

For the word-similarity evaluation script you will need:

- Python 2.6 or newer
- NumPy & SciPy

For the python bindings (see the subdirectory python) you will need:
- Python version 2.7 or >=3.4
- NumPy & SciPy
- pybind11
